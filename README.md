# CloudType_study

- 작업 진행 후 front 파일에 npm run build
- front 파일에 생성된 build 파일 ctrl + c
- back 파일에 ctrl + v
- 깃 허브 레파지토리 생성 후 back파일 업로드
    - 업로드시 .env 파일과 node_modules 파일 제외
    - (분홍 형광펜 표시만 올리기)
